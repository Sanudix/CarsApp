# Android-приложение "Автомобильный маркетплейс"

## Описание
Это проект, написанный для учебных целей. Для упрощения работы с базой данных был использован Firebase. Весь UI создан с помощью Jetpack Compose. Приложение построено с помощью архитектурного паттерна MVVM.

Стек:
- Kotlin
- Jetpack Compose
- Архитектурный паттерн MVVM
- Firebase
